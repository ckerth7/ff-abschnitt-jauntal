Description:
Different representations (in terms of size) are used for each image to support responsive behaviour. 
This node.js app is used to create those representations based upon a folder of input images.

How to use the app:
1. Download and install node.js
2. Open "node.js command promt" from the start menu
2. Wait until the promt opens and enter "npm install sharp"
3. Open the cmd in the same directory "app.js" is located
4. Issue "node app.js" command
